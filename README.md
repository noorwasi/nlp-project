# NLP Master Project — Sentiment Analysis of Movie Reviews

## Project Overview
This project is part of the NLP Master course. The goal is to build a
text classification system for sentiment analysis on the IMDb Movie
Reviews dataset (50k reviews, binary sentiment: positive/negative).

The project is divided into two phases:
- **Baseline Phase (Weeks 1–6):** Build a classifier from scratch in Python with no ML libraries
- **Specialization Phase (Weeks 6–11):** Improve using modern libraries (Transformers, LLMs, RAG)

## Dataset
- **Name:** IMDb Dataset of 50k Movie Reviews
- **Source:** [Kaggle](https://www.kaggle.com/datasets/lakshmi25npathi/imdb-dataset-of-50k-movie-reviews)
- **Format:** CSV with two columns — `review` and `sentiment`
- **Labels:** `positive` / `negative`
- **Subset used for baseline:** 2000–5000 samples
